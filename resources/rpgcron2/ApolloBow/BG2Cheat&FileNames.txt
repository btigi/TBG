Baldur's Gate 2

First: You need to enter Debug Mode=1 under Program Options in the baldur.ini file in your Baldur's Gate directory and then save.
Second: Import the .tbg with the Infinity Addon Patcher after unzipping the .zip file.
Third: Start Baldur's Gate 2 and press CTRL+Space while in the game.
Fourth: Type one of the following exactly as I have it, without the name of the item.
